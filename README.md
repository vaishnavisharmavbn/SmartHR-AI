# SmartHR-AI

An AI-Powered HR Management System built using Flask and Python.

## Features

- Employee Management
- HR Dashboard
- Attendance Management
- Authentication System
- Light & Dark Mode
- Responsive UI
- AI-based HR Enhancements (Coming Soon)

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Installation

```bash
git clone https://github.com/vaishnavisharmavbn/SmartHR-AI.git
cd SmartHR-AI
pip install -r requirements.txt
python app.py
```
